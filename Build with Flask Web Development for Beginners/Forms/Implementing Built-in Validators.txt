WTForms provides several built-in validators that you can use to validate form data. Some common validators include:

DataRequired(): Ensures the field is not empty
Length(min, max): Checks if the input length is within a specified range
Email(): Verifies if the input is a valid email address
NumberRange(min, max): Checks if a number is within a specified range






To use these validators, import them from wtforms.validators and add them to your form fields:

from flask_wtf import FlaskForm
from wtforms import StringField, PasswordField
from wtforms.validators import DataRequired, Email, Length

class RegistrationForm(FlaskForm):
    username = StringField('Username', validators=[DataRequired(), Length(min=4, max=20)])
    email = StringField('Email', validators=[DataRequired(), Email()])
    password = PasswordField('Password', validators=[DataRequired(), Length(min=8)])






Handling Validation Errors: When processing form submissions in your route, you can check if the form is valid using the validate_on_submit() method:

@app.route('/register', methods=['GET', 'POST'])
def register():
    form = RegistrationForm()
    if form.validate_on_submit():
        # Process valid form data
        return 'Registration successful!'
    return render_template('register.html', form=form)



If validation fails, you can access error messages through form.errors in your template:

{% for field, errors in form.errors.items() %}
    <div class="error">
        {{ form[field].label }}: {{ ', '.join(errors) }}
    </div>
{% endfor %}

from flask import Flask, render_template, request, redirect, url_for
from wtforms import Form, StringField, SubmitField, IntegerField
from wtforms.validators import InputRequired, Email, NumberRange

app = Flask(__name__)

class NameForm(Form):
    name = StringField('Name', validators=[InputRequired()])
    age = IntegerField('Age', validators=[InputRequired(), NumberRange(min=1, max=120)])
    submit = SubmitField('Submit')

@app.route('/', methods=['GET'])
def show_form():
    form = NameForm()
    return render_template('register.html', form=form)

@app.route('/submit', methods=['POST'])
def handle_submit():
    form = NameForm(request.form)
    if form.validate():
        name = form.name.data
        age = form.age.data
        return f'<h1>Hello, {name}! and you are {age} years old.</h1>'
    return render_template('register.html', form=form)

if __name__ == '__main__':
    app.run(debug=True)
